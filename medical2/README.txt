# Medical Expert - Site Institucional

## Como usar
Coloque todos os arquivos deste diretório em um servidor local (ex: XAMPP, WAMP) ou em um servidor web.

## Estrutura de arquivos
- index.html: Página principal do site
- style.css: Estilos do site
- README.txt: Este arquivo
- As imagens abaixo devem ser adicionadas na mesma pasta:
  - logo-medical-expert.png (logo principal)
  - ricardo-azevedo.jpg (foto do fundador)
  - cellex.png (produto Cellex)
  - jj-cannula.png (produto Johnson & Johnson)
  - welld.png (produto Welld)
  - innolcon.png (produto Innolcon)
  - whatsapp.svg (ícone do WhatsApp)
  - bg-medical.jpg (imagem de fundo para o topo)

## Personalização
- Edite o arquivo index.html para alterar textos, produtos ou membros da equipe.
- O botão do WhatsApp pode ser alterado no href do link (linha com 'wa.me/5511999999999').

## Responsividade
O site é totalmente responsivo e se adapta a celulares, tablets e desktops.
